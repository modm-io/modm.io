    modm:platform:gpio¶

    Options¶

    enable_ports¶

    Enable clock for these GPIO ports during startup: ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J', 'K'] ∈ { A, B, C, D, E, F, G, H, I, J, K }
